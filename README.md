# Transport Layer Security (TLS)

is the factual standard for authenticity, privacy, and integrity of data in transit.

# Mojo

in [Phase 1: High performance CPU + GPU coding](https://mojolang.org/docs/roadmap/#phase-1-high-performance-cpu--gpu-coding)
comes with [language features](https://mojolang.org/docs/reference/),
a [standard library](https://mojolang.org/docs/std/)
and [community packages](https://mojolang.org/packages/).

TLS is a very specific feature and actually specified in standards like [IETF RFC 8446](https://datatracker.ietf.org/doc/html/rfc8446) "The Transport Layer Security (TLS) Protocol Version 1.3".
Therefore TLS is not implemented directly in the language.

# Safe Artificial Intelligence (AI)

keeps the effects of AI systems within safe limits, defined by the operator.

Large and complex models are not random.
They are deterministic, but can be intractable or unfathomable.

One can run the model in a defined space.
A positive list of skills enables the model to interact.
Receiving input is a skill and emitting output is a skill.
Both can be filtered with socalled guardrails.

The input filter is the firewall and the input validation for AI.

The output filter keeps the effects in the defined safe zone, if there is a fail-safe mode.

These two filters can be controlled by the runtime or virtual machine.

Container virtualization can start different inference engines and models.
